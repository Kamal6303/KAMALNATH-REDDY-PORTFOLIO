Image Forgery Detection

Developed a Digital Image Forgery Detection system using CNN and Error Level Analysis (ELA). The system allows users to upload images through a web interface and analyzes them for potential forgery. A CNN-based model classifies images as authentic or manipulated, enhancing detection accuracy. Optimized image preprocessing and model performance for reliable and efficient forgery detection.

Stock Market Analysis

Developed a Python-based application that collects and analyzes historical stock data using Pandas, NumPy, and yFinance API. Implemented machine learning and LSTM models to forecast future stock prices, applied technical indicators (SMA, EMA, RSI) for trend analysis, and visualized insights using Matplotlib and Seaborn.
